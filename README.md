**Snes9x** is a well-established emulator for the Super Nintendo Entertainment System, known for its strong compatibility, accuracy, and consistent performance.

This fork adapts Snes9x to work with the Tico frontend and provides a standalone build for the Nintendo Switch, adding a small set of practical features while preserving the qualities that made the original a trusted choice.

----------

## Summary

This fork focuses on making Snes9x more usable in practice without changing its core design.

It adds:

-   Custom overlay matching Tico design, including time, date, user avatar, and game title
-   Explicit control over display (integer scaling and aspect ratios)
-   Runtime-selectable rendering filters  
-   Built-in save and load state support
-   Integrated RetroAchievements with custom alerts
